## Buscador de Noticias

Esta app permite buscar noticias en EE.UU por selección de categoria. También utiliza una paginación para subdividirlas en páginas.

## Herramientas

- React/ Javascript
- Material UI
- Hooks

## LEEME

Clonar Repositorio

```
$ git clone [repo url]
```

Instalar Dependencias

```
$ npm install
```

Correr Servidor de prueba

```
$ npm run dev
```
